# duilib
https://github.com/duilib/

当前项目以duilib v1.1版本为基础，增加了CNTVDemo，ExplorerDemo和WeChatDemo工程，
包括CNTVRes，SogouExplorerRes和WeChatRes等资源文件，bin目录中包含exe程序。

特别说明：
	新增加的工程对应的资源文件仅供学习参考之用，其中：
	（1）CNTVRes下的所有文件均来自央视影音程序；
	（2）SogouExplorerRes下的除xml文件外，其他均来自搜狗浏览器程序；
	（3）WeChatRes下除xml文件，背包头像图片外，其他均来自微信PC版程序。
	故如若用作商业用途，使用前请咨询相关版权方，本人不承担相关之责任。
